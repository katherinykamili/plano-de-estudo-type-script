Apresentação

O que é TypeScript?

TypeScript é uma linguagem criada sobre o JavaScript. Isso significa que todo código JavaScript válido pode ser usado em TypeScript, mas o TypeScript acrescenta um recurso muito importante: tipos.

Os tipos ajudam você a explicar ao computador, ao editor e a outras pessoas que tipo de dado uma variável, função ou objeto deve receber. Em projetos web, isso reduz erros, melhora a leitura do código e torna a manutenção mais segura.

Conteúdo programático

Explicação simples, exemplo e prática

1. Tipos primitivos

Conceito: tipos básicos como string, number e boolean.

Para que serve: evitar que uma variável receba um valor inesperado.

let nome: string = "Ana";
let idade: number = 20;
let estudando: boolean = true;

Exercício: crie três variáveis tipadas com seu nome, sua idade e se você já estudou JavaScript.

2. Funções tipadas

Conceito: funções podem ter tipos nos parâmetros e no retorno.

Para que serve: deixar claro o que a função recebe e devolve.

function somar(a: number, b: number): number {
  return a + b;
}

Exercício: crie uma função que receba o nome de um curso e retorne uma mensagem de boas-vindas.

3. Objetos, interfaces e types

Conceito: interfaces e types descrevem o formato de um objeto.

Para que serve: padronizar dados, como aluno, produto ou conteúdo estudado.

interface Aluno {
  nome: string;
  nivel: "iniciante" | "intermediario" | "avancado";
}

Exercício: crie uma interface para representar um livro de estudos.

4. Generics

Conceito: generics criam funções reutilizáveis que preservam o tipo recebido.

Para que serve: evitar repetição sem perder segurança de tipos.

function primeiroItem<T>(lista: T[]): T {
  return lista[0];
}

Exercício: crie uma função generic que receba uma lista e retorne o último item.

TypeScript na prática

Exemplos para ler, copiar e testar

Variáveis e funções

const curso: string = "TypeScript";

function criarMensagem(nome: string): string {
  return `Olá, ${nome}! Vamos estudar ${curso}.`;
}

Objeto com interface

interface Produto {
  nome: string;
  preco: number;
}

const livro: Produto = {
  nome: "Guia de TypeScript",
  preco: 59.9
};

Classe

class Aluno {
  constructor(public nome: string) {}

  apresentar(): string {
    return `${this.nome} está aprendendo TypeScript.`;
  }
}

Generic

function listar<T>(itens: T[]): T[] {
  return itens;
}

listar<string>(["Tipos", "Funções", "Classes"]);

Exercícios

Pratique por nível

Básico

Enunciado: crie variáveis tipadas para nome, idade e status de matrícula.

Dica: use string, number e boolean.

Objetivo: entender tipos primitivos.

Intermediário

Enunciado: crie uma interface Conteudo e uma função que liste conteúdos concluídos.

Dica: use arrays e literal types para o status.

Objetivo: organizar objetos com contratos de tipo.

Avançado

Enunciado: crie uma função generic para buscar itens por propriedade.

Dica: pesquise keyof e generics com restrições.

Objetivo: criar funções reutilizáveis com segurança.

Projeto final

Sistema de Cadastro de Estudos em TypeScript

O projeto final será um sistema simples para registrar conteúdos estudados. A ideia é praticar tipos, interfaces, funções e arrays em um contexto real.

  • Cadastro de conteúdos estudados.
  • Nível de dificuldade: básico, intermediário ou avançado.
  • Status: não iniciado, em andamento ou concluído.
  • Lista de estudos renderizada na tela.
  • Uso de tipos, interfaces, funções e arrays.
type Status = "não iniciado" | "em andamento" | "concluído";

interface Estudo {
  titulo: string;
  dificuldade: "básico" | "intermediário" | "avançado";
  status: Status;
}

Ferramentas e passo a passo

Recursos necessários

  • VS Code

  • Node.js

Passo a passo

  1. Clone ou baixe este projeto.
  2. Abra a pasta no VS Code.
  3. Instale o TypeScript com npm install -g typescript.
  4. Compile arquivos .ts com tsc typescript/basico.ts.
  5. Abra o index.html no navegador.
  6. Publique no GitHub Pages pela aba Settings > Pages.
