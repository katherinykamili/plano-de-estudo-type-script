# Guia de Estudos

Use este arquivo como apoio para acompanhar o site.

## Ordem recomendada

1. Leia a apresentação para entender o papel do TypeScript.
2. Estude a ementa para visualizar o mapa do curso.
3. Entre na seção "Matérias passo a passo".
4. Clique em um tema e leia a explicação no pop-up.
5. Siga os passos indicados na matéria.
6. Copie o exemplo para um arquivo `.ts`.
7. Faça a atividade prática proposta.
8. Resolva os exercícios por nível.
9. Finalize com o Sistema de Cadastro de Estudos em TypeScript.

## Como usar os pop-ups de matéria

Cada pop-up tem quatro partes:

- Explicação: apresenta o conceito com linguagem simples.
- Passo a passo: mostra uma ordem segura para estudar.
- Exemplo de código: oferece uma base para copiar e testar.
- Atividade prática: transforma a teoria em exercício.

## Dica para iniciantes

Não tente decorar todos os tipos de uma vez. Primeiro entenda por que eles existem: tipos ajudam a prevenir erros antes de o código rodar no navegador.
